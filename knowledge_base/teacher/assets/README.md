# 教师示例资源

本目录存放教师线为课堂准备的示例图。

当前资源:
- `L002_primitives_sample.png`: 立方体/圆柱/球体的结构示例。
- `L002_combo_sample.png`: 2-3形体组合的结构稿与轮廓稿对照示例。
- `L003_sample_sheet.png`: 一点透视可视示例（课堂展示）。
- `L003_sample_sheet.md`: 一点透视标准示例+错例纠偏脚本。
- `L004_sample_sheet.png`: 三值塑形可视示例（课堂展示）。
- `L004_sample_sheet.md`: 三值塑形标准示例+脏灰纠偏脚本。
- `L005_sample_sheet.png`: 二点透视可视示例（课堂展示）。
- `L005_sample_sheet.md`: 二点透视道具结构标准示例+错例纠偏脚本。

使用方式:
- 学员线开课时先展示PNG示例，再进入跟练。
- 教师讲解时可配合同名MD脚本进行口播与纠偏提示。
- 独立小测阶段不展示示例，避免只会照抄。
- 每课至少准备1个标准示例+1个反例，避免课堂临时补素材。
