## 1) 本轮美术学习纪要（教学化提炼）
- 一点透视初学者高频错在"退向线不回同一消失点"，必须把"同点收敛"作为单一核心检查项先打稳。
- 二点透视初学者高频错在"近角不稳定"，若近角漂移，后续所有边都会塌；因此先立近角再连消失点更稳定。
- 三值塑形里，体积感的关键不是平滑程度，而是亮中暗关系是否先被拉开；先大关系后细节可显著减少脏灰。
- 道具结构教学应固定"结构版 -> 轮廓版"顺序，避免学员直接描外形导致可读性崩溃。

## 2) 本轮教育学习纪要（教学法提炼）
- 对0基础学员，新概念统一用"是什么 -> 为什么 -> 怎么做 -> 怎么判对错"四句脚本，可降低理解门槛。
- Worked Example需要与Fading配套: 前半程给标准示例，后半程必须撤掉样例做限时独立迁移。
- 课堂等待是强干扰项，素材和示例应在教师线预准备，学员线只执行不临时搜集。
- 反馈格式固定为"1个关键问题 + 1个可执行改法"，可避免一次性信息过载。
