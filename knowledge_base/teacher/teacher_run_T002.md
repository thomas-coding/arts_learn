## 1) 本轮美术学习纪要（教学化提炼）
- 形体训练从"单体"过渡到"组合体"时，初学者最容易在遮挡关系和轴线方向上崩溃，必须先做少量高质量组合，不追数量。
- 椭圆训练不能只追闭合，关键是"长短轴一致+倾角一致"；同组训练应保持同一倾角，降低变量。
- 透视入门里，二点透视比一点透视更容易暴露"平行边不平行"和"错误收敛"问题，适合作为结构稳定性的二次检验课。
- 物体拆解教学建议固定三步: 包围盒 -> 基本体块 -> 外轮廓清理，可显著降低"上来画细节"的错误概率。

## 2) 本轮教育学习纪要（教学法提炼）
- 显性教学（Explicit Instruction）: 对0基础学员，术语与动作必须显式定义，先给"什么是对"再练习，避免猜。
- 范例效应（Worked Example）: 课堂前半程给标准样例可降低认知负荷，但后半程必须切换独立迁移题，避免只会临摹。
- 检索练习（Retrieval Practice）: 在课堂末段让学员离开示例独立完成，可提升长期记忆与稳定复现。
- 掌握性学习（Mastery Learning）: 关键指标未达标时插入补强微课，比硬推下一课更高效。
