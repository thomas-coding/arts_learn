## 1) 本轮美术学习纪要（教学化提炼）
- 线条控制优先级高于精细造型；初学者先建立"稳定笔触"再追求细节准确。
- 从2D到3D的过渡路径应为: 基础形状 -> 体块 -> 透视 -> 明暗，而不是直接临摹复杂图。
- 初学者最常见问题是"看见细节就画细节"，需强制先做大形和包围盒。
- 透视入门不求复杂理论，先让学员掌握地平线、消失点、收敛方向三件事。
- 明暗入门先做3值（亮/中/暗），再进入更多灰阶，能显著降低脏灰风险。

## 2) 本轮教育学习纪要（教学法提炼）
- 认知负荷管理: 每节课只设1个核心能力，避免一次塞入过多概念。
- 脚手架: 教师示范 -> 跟练 -> 独立小测 -> 反馈，确保可迁移而非只会照抄。
- 形成性评价: 当堂必须有可量化Rubric，不把验收推迟到课后。
- 刻意练习: 课后作业拆成短时高频小练，强调重复与即时纠偏。
- 反馈策略: 先指出1个最关键错误，再给1个可执行改法，避免信息过载。

## 4) 教学风险与预案
- 风险A: 学员急于画复杂对象 -> 预案: 强制先做包围盒和形体分解。
- 风险B: 课堂内完成度低 -> 预案: 缩减对象数量，保留核心动作不变。
- 风险C: 作业拖延 -> 预案: 课后任务拆成20-30分钟可完成单元。
