# 24个月课程总地图（讨论冻结版）

说明: 本路线图来自当前协商结果，用于长期方向对齐；具体单节课内容由教师线动态备课。

## 阶段1（M1-M6）基础造型与视觉语言
- M1: 观察与线条控制。
- M2: 透视与体块。
- M3: 明暗与体积。
- M4: 色彩基础。
- M5: 材质表达。
- M6: 构图与叙事。
- Gate A: 基础能力闸门。

## 阶段2（M7-M12）角色方向打底
- M7: 动态与人体简化。
- M8: 头部/五官/发型设计。
- M9: 服装与形状语言。
- M10: 角色立绘流程。
- M11: 武器与道具设计。
- M12: 角色包v1整合。
- Gate B: 角色方向闸门。

## 阶段3（M13-M18）场景/UI/特效与管线整合
- M13: 场景透视与镜头。
- M14: 模块化场景资产。
- M15: 风格化场景深化。
- M16: UI图标与HUD。
- M17: 2D特效与简动效。
- M18: 纵切片v1（角色+场景+UI+特效）。
- Gate C: 管线整合闸门。

## 阶段4（M19-M24）完整项目冲刺与作品集
- M19: 预制作（风格圣经/资产清单）。
- M20: 冲刺1（角色/NPC）。
- M21: 冲刺2（场景/道具/UI）。
- M22: 冲刺3（特效/宣发图）。
- M23: 全量统一与打磨。
- M24: 最终封包与作品集。
- Gate D: 最终项目闸门。

## 闸门规则
- 任一Gate未达标: 先补强再进入下一阶段。
- 验收以"可用于项目"为核心标准，而非仅"看起来像"。
