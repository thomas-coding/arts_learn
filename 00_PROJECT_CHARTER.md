# 2D游戏美术长期教学项目协商纪要（锁定版）

版本: v1.0
状态: 已生效（用于防止context过期后的信息漂移）
单一事实源: 本文件优先级最高；若与其他文件冲突，以本文件为准。

## 1) 项目目标（协商一致）
- 学员起点: 0基础、无美术基础。
- 长期目标: 学员可独立完成一整套2D游戏美术（风格目标: 类原神/王者荣耀气质的2D化表达）。
- 训练周期: 长期项目，可能持续多年。

## 2) 教学运行方式（协商一致）
- 不采用固定排课频率（不绑定每周几次课、几次练习）。
- 采用口令驱动的双线流程:
  - 口令A: "你学习下" -> 进入教师线。
  - 口令B: "我来学习" -> 进入学员线（1次60分钟课程）。

## 3) 教师线规则（协商一致）
- 当收到"你学习下"时，教师线必须执行:
  1. 学习美术相关知识（造型/色彩/构图/角色/场景/UI/特效等）。
  2. 学习教育相关知识（教学法、反馈法、评估法、脚手架等）。
  3. 备课并更新验收标准。
- 备课库存要求:
  - 每次教师线至少准备4节可授课内容:
    - 1节: 下一次可直接上的课程。
    - 3节: 额外预备课（你要求的"每次额外多备3次课"）。

## 4) 学员线规则（协商一致）
- 当收到"我来学习"时，学员线必须执行:
  1. 安排并执行1节60分钟课程。
  2. 全程由教师把控流程，学员按step-by-step执行。
  3. 课程结束后给出课后练习与验收标准。
  4. 验收由教师负责并记录结果。
- 前置条件:
  - 学员上课前，教师应处于已完成学习与备课状态。

## 5) 1小时课程SOP（协商一致）
- 0-8分钟: 目标与上节复盘。
- 8-18分钟: 教师示范与关键原理。
- 18-40分钟: 学员跟练，教师实时纠偏。
- 40-52分钟: 限时独立任务（当堂迁移测试）。
- 52-60分钟: 教师验收评分 + 课后练习布置。

## 6) 双知识库要求（协商一致）
- 必须维护两套知识库:
  - 教师知识库: 美术知识、教育知识、教案、SOP、评分规则、错误库。
  - 学员知识库: 学习进度、课程记录、作业成绩、问题清单、后续规划。
- 每次教师线或学员线执行后，都要同步更新对应知识库。

## 7) 阶段路线图（当前讨论版）
- 0-6个月: 基础造型与视觉语言。
- 7-12个月: 角色方向打底（向目标风格迁移）。
- 13-18个月: 场景/UI/特效与游戏管线整合。
- 19-24个月: 完整项目冲刺与作品集成型。
- 质量闸门: Gate A/B/C/D，未达标先补强再进入下一阶段。

## 8) 反漂移约束
- 不得私自改动以下关键约束:
  - 非固定排课，改为口令驱动。
  - 教师线每次要学习并额外备3节课。
  - 学员线每次1小时、教师主导、当堂验收。
  - 双知识库持续维护。
- 如需改动，必须由用户明确提出并在本文件记录版本变更。
